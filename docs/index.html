Get Started in 60 Seconds

git clone https://github.com/alexandrebrg/fig-revival.git
cd fig-revival
./install.sh
1

Clone & Install

Clone the repository and run the installation script. It will build the binaries and set up shell integration.

2

Restart Your Shell

Restart your terminal or run exec zsh (or exec bash) to load the integration.

3

Start Typing

That's it! Start typing commands and enjoy intelligent autocomplete suggestions.
